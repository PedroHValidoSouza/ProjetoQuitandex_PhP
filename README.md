#  Quitandex :shopping_cart:

### :mag_right: Sobre o projeto:
**Quitandex** é um software de listagem de compras de supermercado nascido de uma ideia para um projeto integrador realizado por uma equipe de estudantes da instituição Fatec[^1]. Rafael Almeida Camarinha, localizada em Marília-SP.
